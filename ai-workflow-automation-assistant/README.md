# 🤖 AI Workflow Automation Assistant

An AI-powered productivity assistant designed to help users transform business text into organized summaries, task lists, professional emails, action plans, and priority analysis.

---

# 🚀 Features

✅ Generate business summaries  
✅ Create structured task lists  
✅ Draft professional emails  
✅ Build action plans  
✅ Analyze task priorities  

---

# 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- AI Workflow Concepts
- Prompt Engineering Concepts

---

# 📌 Project Overview

This project demonstrates how AI-inspired workflow automation can improve productivity and support business operations by organizing unstructured text into actionable outputs.

The assistant allows users to paste meeting notes, requests, or business messages and generate structured results instantly.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/22-dhay/my-portfolio.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
